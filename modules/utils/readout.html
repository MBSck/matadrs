matadrs.utils.readout module#

class matadrs.utils.readout.ReadoutFits(fits_file: Path, flux_file: Path | None = None)[source]#

Bases: object

Reads out the Cards as Tables as well as the primary header of a (.fits)-file and makes certain keys from the primary header available as properties of the class.

Parameters:

  • fits_file (pathlib.Path) – The (.fits)-file from which data is sourced.

  • flux_file (pathlib.Path, optional) – Additional flux file, that replaces the total flux data for the (.fits)-file.

fits_file#

The (.fits)-file that has been read in.

Type:

pathlib.Path

flux_file#

If provided, substitutes the columns of the ‘oi_flux’ Table with the values from the flux file.

Type:

pathlib.Path, optional

primary_header#

The primary header of the (.fits)-file.

Type:

astropy.io.fits.PrimaryHDU
