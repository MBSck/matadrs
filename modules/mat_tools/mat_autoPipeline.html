matadrs.mat_tools.mat_autoPipeline.mat_autoPipeline(dirRaw='', dirResult='', dirCalib='', nbCore=0, resol=0, paramL='', paramN='', overwrite=0, maxIter=0, skipL=0, skipN=0, tplstartsel='', tplidsel='', spectralBinning=[]) → None[source]#

Runs the esorex MATISSE pipeline command line script with the correct settings.

Parameters:

  • dirRaw (str, optional) – The directory containing the raw files.

  • dirResult (str, optional) – The directory to contain the reduced files.

  • dirCalib (str, optional) – The directory containing the with the observation associated calibration files.

  • nbCore (int, optional) – The number of cores used for the reduction.

  • resol (int, optional) – The resolution of the observation.

  • paramL (str, optional) – A string containing the parameters for the command line script for the L-band.

  • paramN (str, optional) – A string containing the parameters for the command line script for the N-band.

  • overwrite (int, optional) – If toggled, the previous reduction is overwritten.

  • maxIter (int, optional) – The number of iterations done within the reduction.

  • skipL (int, optional) – If toggled, the L-band will be skipped.

  • skipN (int, optional) – If toggled, the N-band will be skipped.

  • tplstartsel (str, optional) – The time stamp start which is to be reduced.

  • tplidsel (str, optional) – The time stamp which is to be reduced.

  • spectralBinning (list, optional) – The spectral binning corresponding to the spectral resolution of the observation.
