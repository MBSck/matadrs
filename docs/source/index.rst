The (modular) MATISSE Automated Data Reduction Software (:bash:`matadrs`) is a tool written to
encompass all the individual steps and files needed to procure and reduce data generally
and easily in as few steps as possible with the least knowledge required.

To make it possible for seasoned MATISSE users to quickly and easily
reduce data and giving others who are interested in MATISSE data the
opportunity to do so as well.
