============================
matadrs.utils.options module
============================

This module contains a dictionary that is used by many parts of
the :bash:`matadrs` package.

Plot Settings
=============

These are the settings for the :mod:`matadrs.utils.plot` module.

.. code-block:: python

   options["colors"] = ["b", "g", "r", "c", "m", "y"]
   options["linestyles"] = ["-", "--", "-.", ":"]
